# Exercise 08: Cybersecurity Workflow

## Goal

Create a sanitized lab journal entry.

## Tasks

1. Create `labs/ssh-hardening.md`.
2. Include goal, environment, commands, observations, and lessons learned.
3. Avoid real hostnames, usernames, IP addresses, and credentials.
4. Commit the lab note.
5. Open the file on GitHub and read it as a portfolio reviewer.

## Reflection

- Does the writeup show skill without exposing sensitive data?
- What would you move to a private repo?
